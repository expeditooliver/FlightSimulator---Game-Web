# Screenshots do FlightSimulator

Este diretório armazena imagens usadas no `README.md` do projeto.

Arquivos esperados (adicione aqui seus PNGs/JPGs com estes nomes):

- `landing.png` — Tela inicial do simulador
- `controls.png` — Janela de controles e dicas de voo
- `inflight.png` — Vista em perseguição durante o voo

Observações:
- Mantenha resolução razoável (largura entre 1200–1600 px) para boa visualização no GitHub.
- Prefira PNG para nitidez; JPG para tamanhos menores.
- Caso renomeie os arquivos, atualize os caminhos no `README.md`.
